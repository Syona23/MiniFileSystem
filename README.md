# Mini File System Simulator (C++)

A command-line file system simulator built using C++ and data structures.

## Features
- mkdir – create directory
- touch – create file
- ls – list contents
- cd – change directory
- pwd – show current path
- rm – delete file
- tree – show directory structure
- find – search files
- rename – rename files
- du – disk usage

## Data Structures Used
- N-ary Tree for directory hierarchy
- unordered_map for fast lookup
- DFS traversal for file search

## Compile

g++ main.cpp FileSystem.cpp -o filesystem

## Run

./filesystem
